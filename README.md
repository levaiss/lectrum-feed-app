# Професійний курс по React, Redux, MobX, TypeScript

1. Install dependencies:

```sh
yarn
# или
npm i
```

2. Run dev mode:

```sh
yarn dev
# или
npm run dev
```

3. Build prod:

```sh
yarn build
# или
npm run build
```

4. For run in `production` mode:

```sh
yarn build && yarn start  
# или
npm run build && npm run start
```
