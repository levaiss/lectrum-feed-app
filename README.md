# Professional course on React, Redux, MobX, TypeScript

1. Install dependencies:

```sh
yarn
# или
npm i
```

2. Run development server

```sh
yarn dev
# или
npm run dev
```

3. Build for prod:

```sh
yarn build
# или
npm run build
```

4. Run server in `production` mode:

```sh
yarn build && yarn start  
# или
npm run build && npm run start
```
